## Kurulum

### Sistem Gereksinimleri

| Gereksinim | Minimum |
|------------|---------|
| Isletim Sistemi | Windows 10/11 (64-bit) |
| RAM | 4 GB |
| Disk Alani | 500 MB |
| Ekran Cozunurlugu | 1366x768 |

### Kurulum Adimlari

1. **Indirme:** [takibiesasi.com/indir](https://takibiesasi.com/indir) adresinden kurulum dosyasini indirin.

2. **Calistirma:** Indirilen `TakibiEsasi-Setup.exe` dosyasina cift tiklayin.

3. **Lisans Sozlesmesi:** Kullanim sartlarini okuyup kabul edin.

4. **Kurulum Yeri:** Varsayilan konum onerilir: `C:\Program Files\TakibiEsasi`

5. **Tamamlama:** "Yukle" butonuna tiklayin ve kurulumun tamamlanmasini bekleyin.

### Ilk Calistirma

Kurulum sonrasi uygulama otomatik baslar. Ilk acilista:

- **Demo Modu:** 14 gun ucretsiz deneme
- **Lisans Girisi:** Zaten lisansiniz varsa bu secenegi kullanin

## Ilk Adimlar

### Giris Ekrani

Ilk giriste varsayilan bilgiler:
- **Kullanici Adi:** admin
- **Sifre:** admin

**Guvenlik:** Ilk giristen sonra sifrenizi mutlaka degistirin!

### Ana Ekran Tanitimi

```
+----------------------------------------------------------+
| [Logo] TakibiEsasi                   [Kullanici] [Ayar]  |
+------------+---------------------------------------------+
|            |                                             |
| Dosyalar   |  [Dosya Listesi]                           |
| Finans     |                                             |
| Raporlar   |                                             |
| Arsiv      |                                             |
| Ayarlar    |                                             |
|            |                                             |
+------------+---------------------------------------------+
```

### Yeni Dosya Olusturma

1. "Dosyalar" sekmesine gidin
2. "Yeni Dosya" butonuna tiklayin
3. Gerekli bilgileri doldurun:
   - Dosya numarasi (otomatik veya manuel)
   - Muvekkil bilgileri
   - Karsi taraf
   - Dava turu
   - Mahkeme bilgileri
4. "Kaydet" butonuna tiklayin

## Dosya Yonetimi

### Dosya Arama

Ust kisimdaki arama kutusunu kullanarak:
- Dosya numarasi
- Muvekkil adi
- Karsi taraf adi
- Mahkeme

ile arama yapabilirsiniz.

### Dosya Filtreleme

Sol paneldeki filtreler:
- **Durum:** Acik, Kapali, Beklemede
- **Tur:** Icra, Dava, Danismanlik
- **Tarih Araligi:** Baslangic - Bitis

### Dosya Detaylari

Dosyaya tikladiginizda acilan sekmeler:
- **Genel:** Temel bilgiler
- **Taraflar:** Muvekkil ve karsi taraf detaylari
- **Islemler:** Yapilan islemler ve tarihler
- **Belgeler:** Yuklenen dosyalar
- **Finans:** Mali bilgiler

## Finans Modulu

### Ucret Sozlesmesi

Her dosya icin ucret sozlesmesi tanimlayabilirsiniz:
- Sabit ucret
- Nispi ucret (%)
- Taksit plani

### Tahsilat Kaydi

1. Dosya detayinda "Finans" sekmesine gidin
2. "Tahsilat Ekle" butonuna tiklayin
3. Tutar ve tarihi girin
4. Aciklama ekleyin (opsiyonel)
5. "Kaydet"

### Finansal Raporlar

- Toplam alacak
- Toplam tahsilat
- Bekleyen odemeler
- Aylik/yillik gelir grafigi

## Yedekleme

### Otomatik Yedekleme

Ayarlar > Yedekleme bolumunden:
- Gunluk/Haftalik/Aylik otomatik yedekleme
- Yedekleme konumu secimi
- Yedek sayisi limiti

### Manuel Yedekleme

1. Ayarlar > Yedekleme
2. "Simdi Yedekle" butonuna tiklayin
3. Konum secin
4. Yedekleme tamamlanana kadar bekleyin

### Geri Yukleme

1. Ayarlar > Yedekleme
2. "Geri Yukle" butonuna tiklayin
3. Yedek dosyasini secin
4. Onaylayin

**Dikkat:** Geri yukleme mevcut verilerin uzerine yazar!

## Sikca Sorulan Sorular

### Genel

**S: Demo suresi dolunca verilerim silinir mi?**
C: Hayir, tum verileriniz korunur. Lisans satin alarak kaldiginiz yerden devam edebilirsiniz.

**S: Birden fazla bilgisayarda kullanabilir miyim?**
C: Bireysel lisans tek cihaz icindir. Lisans transferi icin destek ile iletisime gecin.

**S: Internet baglantisi gerekli mi?**
C: Hayir, uygulama tamamen offline calisir. Sadece lisans aktivasyonu ve guncellemeler icin internet gerekir.

### Teknik

**S: Verilerim nerede saklaniyor?**
C: Verileriniz bilgisayarinizda sifreli olarak saklanir: `%APPDATA%\TakibiEsasi\data.db`

**S: Uygulamayi baska bilgisayara tasiyabilir miyim?**
C: Evet, yedek alarak ve yeni bilgisayarda geri yukleyerek tasiyabilirsiniz. Lisans transferi icin destek@takibiesasi.com adresinden talep olusturun.
